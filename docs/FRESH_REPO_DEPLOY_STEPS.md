# Fresh repo deploy steps

1. Clear the GitHub repository contents.
2. Upload the contents of this ZIP, not the ZIP itself.
3. Ensure GitHub Pages serves from the root of the main branch.
4. Open the app URL.
5. Sign in.
6. Use **user menu → Import DB** to open `/tools/importer.html`.
7. Upload your cleaned `dead_wax_db.js`.
8. Analyze. The expected Shelf ID count is 559 for the current cleaned baseline.
9. Import.
10. Return to the app and refresh.
